OkenV1AddressRegistry

This contract stores the deployment addresses of Oken V1 contracts.

This contract inherits OpenZeppelin's Ownable contract.

Address​

tokenRegistry​

function tokenRegistry() external view returns (address)

Returns the address of the OkenV1TokenRegistry contract.

NameTypeDescription
[0]addressAddress of the OkenV1TokenRegistry contract

updateTokenRegistry​

function updateTokenRegistry(address newTokenRegistry) external

Modifies the address of the OkenV1TokenRegistry contract.

This function will revert if msg.sender is not the contract owner.

NameTypeDescription
newTokenRegistryaddressAddress of the new OkenV1TokenRegistry contract
